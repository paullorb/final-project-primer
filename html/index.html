HTML

HTML stands for HyperText Markup Language

<!DOCTYPE>
        <html>
            <head>
                <title>Page Title</title>
            </head>
            <body>
                <h1>This is a Heading</h1>
                <p>This is a paragraph.</p>
            </body>
        </html>
<!-- This is a comment -->

They can have attributes inside the <> that modify their behavior.

HTML forms are used to collect user input. Key elements include:

<form>
<input> type="password" name="password"
<!-- this element is optional --> <label for="password"> Enter password</label>
<!-- this one also --> <textarea></textarea>
<!--  --> <button type="submit"></button>
</form>

Forms send data to a server for processing, typically using GET or POST methods.

Tables are used for displaying tabular data.

<table>
    <thead>
        <tr>
            <th>Name</th>
        </tr>
    </thead>
        <tr>
            <td>Paul</td>
        </tr>
<table>

Tables can have headers, footers, and bodies.
